- Python 3

## Getting started 
### Installation 
- The required libraries are matplotlib, numpy and pytorch (add all requirements!)

```bash
pip install -r requirements.txt
```

- Clone this repo:
```bash
git clone https://github.com/mats-granath/toric-RL-decoder.git
cd toric-RL-decoder
```
